DOSSIER MEDIA - INSTRUCTIONS
=============================

Ce dossier contient tous les médias (images, vidéos) utilisés sur le site.

STRUCTURE :
-----------
media/
├── avatars/     → Photos de profil (créateur et utilisateurs)
├── images/      → Photos pour les posts
└── videos/      → Vidéos pour les posts

COMMENT AJOUTER DES MÉDIAS :
-----------------------------

1. AVATARS (media/avatars/)
   - Nommez le fichier principal : creator.jpg
   - Nommez les autres : user1.jpg, user2.jpg, user3.jpg, user4.jpg
   - Format recommandé : JPG ou PNG
   - Taille recommandée : 200x200px minimum

2. IMAGES (media/images/)
   - Nommez vos fichiers : photo1.jpg, photo2.jpg, etc.
   - Format : JPG ou PNG
   - Taille : Aucune restriction (seront adaptées automatiquement)

3. VIDÉOS (media/videos/)
   - Nommez vos fichiers : video1.mp4, video2.mp4, etc.
   - Format : MP4 recommandé
   - Taille : Pas de limite (mais privilégiez des fichiers < 100MB)

4. VIGNETTES VIDÉOS (media/images/)
   - Pour chaque vidéo, créez une vignette
   - Nommez : thumb1.jpg pour video1.mp4, thumb2.jpg pour video2.mp4
   - Format : JPG
   - Taille recommandée : 1280x720px

IMAGES PAR DÉFAUT :
-------------------
Si une image n'est pas trouvée, un avatar coloré avec initiale sera généré automatiquement.
Vous n'êtes donc PAS obligé de créer toutes les images immédiatement.

ASTUCE :
--------
Pour tester rapidement, vous pouvez utiliser des images gratuites de sites comme :
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)
